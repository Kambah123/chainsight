The featured cases use real, widely reported addresses — the Ronin bridge exploiter, vitalik.eth, and Binance's BTC cold wallet — with approximate, reconstructed figures for demonstration. The Solana drainer case is a synthetic composite of real drainer tradecraft, so no real address gets mis-accused. Live on-chain data is the next wiring step.

Ethereum, Solana, and Bitcoin in the demo console. The wiring step uses Etherscan API V2 — one key that unlocks 50+ EVM chains — plus Helius for Solana and Blockstream for Bitcoin.

In the demo, analyst reports are pre-written and served through a stubbed data layer. In the live version, an AI analyst generates them from the same normalized case-file format — the interface doesn't change, only the source does.

The search bar validates any ETH, SOL, or BTC address today and loads the featured cases instantly. Arbitrary addresses hit the stub layer and say so honestly — live any-address lookup ships with the backend wiring.
